# Mini Lead Manager – Flutter Internship Assignment

A beautiful, production-ready lead management app built with Flutter 3.24+, Riverpod 2.x, sqflite, and go_router.

## Features
- Add, view, edit, delete leads
- Status management (New → Contacted → Converted → Lost)
- Real-time search & filtering
- Local SQLite database (persistent)
- Light/Dark theme support
- Clean architecture & modern state management
- Safe navigation (no crashes)

## Tech Stack
- Flutter 3.24+
- Riverpod 2.x (State Management)
- sqflite + path_provider (Local DB)
- go_router (Navigation)
- Material 3 Design

## How to Run
```bash
git clone https://github.com/yourusername/mini_lead_manager.git
cd mini_lead_manager
flutter pub get
flutter run# mini_lead_manager
```
